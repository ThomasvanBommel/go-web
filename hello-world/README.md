# Hello-World
[https://gowebexamples.com/hello-world/](https://gowebexamples.com/hello-world/)

# How to run
Run without building
```bash
make run
```
<br/>

Build
```bash
make build
```
<br/>

Run built code
```bash
bin/main
```
<br/>

Build & Run
```bash
make build && bin/main
```

# Expected output
```
Listening http://localhost:8080
```
