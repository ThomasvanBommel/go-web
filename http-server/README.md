# HTTP Server
[https://gowebexamples.com/http-server/](https://gowebexamples.com/http-server/)

# How to run
Run without building
```bash
make run
```
<br/>

Build
```bash
make build
```
<br/>

Run built code
```bash
bin/main
```
<br/>

Build & Run
```bash
make build && bin/main
```
